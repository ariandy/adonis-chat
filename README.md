## How To Run

1. Clone this repository
   ```
   $ git clone https://github.com/ariandy/adonis-chat.git
   ```
2. Install all depedencies on the package.json
   ```
      cd adonis-chat
      npm install
   ```
3. Run the Adonis server
   ```
   $ adonis serve --dev
   ```
